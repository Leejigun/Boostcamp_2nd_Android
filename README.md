### 과정구성
그룹 구성(5~6명) -> 튜터 배정(4명) -> 학습 프로젝트(3주) ->  실무 프로젝트(5주)

#### 학습 프로젝트
* 필수 기초지식 함양
* 자기주도 학습 능력 함양
* 단계별 미니 프로젝트를 통한 코딩 경험 확보


#### 실무 프로젝트
* 프로젝트 사이클 경험
* 프로젝트를 통한 실전 경험 및 실력 향상
* 맞춤형 멘토링을 통한 문제 해결 능력 함양
